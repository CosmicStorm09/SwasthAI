# SwasthAI – AI Community Health Assistant

Concept-stage project for YUVAi Global Youth Challenge (Empowering People & Communities).

## Problem
People hesitate to discuss health concerns due to stigma or lack of access, leading to delayed diagnosis.

## Solution (Concept)
A private, multilingual assistant that:
- Understands symptom descriptions in everyday language
- Returns safe, first-level guidance and red-flag warnings
- Suggests nearby public healthcare options
- Does **not** replace doctors (clear medical disclaimer)
